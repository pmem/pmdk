#### The librpma library

The **Remote Persistent Memory Access (RPMA)** (librpma) is a C library to simplify accessing persistent memory devices on remote hosts over **Remote Direct Memory Access (RDMA)**.

Documentation is available:

See the [librpma man page](../rpma/master/librpma.7.html)
for current master documentation and examples.
